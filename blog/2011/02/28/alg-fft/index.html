еҝ«йҖҹеӮ…йҮҢеҸ¶еҸҳжҚўи®Ўз®—еӨ§ж•ҙж•°д№ҳжі•

2011-02-28 18:56:00

жҲ‘д»¬зҹҘйҒ“пјҢдёӨдёӘ N дҪҚж•°еӯ—зҡ„ж•ҙж•°зҡ„д№ҳжі•пјҢеҰӮжһңдҪҝз”Ёеёёи§„зҡ„з®—жі•пјҢж—¶й—ҙеӨҚжқӮеәҰжҳҜ O(N2)гҖӮ

з„¶иҖҢпјҢдҪҝз”Ёеҝ«йҖҹеӮ…йҮҢеҸ¶еҸҳжҚўпјҢж—¶й—ҙеӨҚжқӮеәҰеҸҜд»ҘйҷҚдҪҺеҲ° O(N logN loglogN)гҖӮ

еҒҮи®ҫжҲ‘д»¬иҰҒи®Ўз®—д»ҘдёӢдёӨдёӘ N дҪҚж•°еӯ—зҡ„д№ҳз§Ҝпјҡ

е°ҶдёҠйқўдёӨдёӘејҸеӯҗзӣёд№ҳпјҢеҫ—еҲ°д»ҘдёӢе…¬ејҸ (е…ұ 2N - 1 йЎ№)пјҡ

йқһеёёе®№жҳ“йӘҢиҜҒпјҢдёҠејҸдёӯзҡ„ ck ( 0 вүӨ k вүӨ 2N-2 ) ж»Ўи¶ід»ҘдёӢе…¬ејҸпјҡ

дёҠејҸе…ұжңү N йЎ№пјҢai е’Ң bj зҡ„дёӢж Ү i е’Ң j ж»Ўи¶і i + j = kгҖӮиӢҘдёҚж»Ўи¶і 0 вүӨ i, j вүӨ N-1 ж—¶пјҢеҲҷд»Ө ai = bj = 0гҖӮ

жҲ‘д»¬д»ҘдёӨдёӘ 3 ( N = 3 ) дҪҚж•° a = 678 е’Ң b = 432 зҡ„д№ҳз§ҜжқҘиҜҙжҳҺд»ҘдёҠиҝҮзЁӢеҗ§гҖӮ

з”ұжӯӨпјҡ

жңҖеҗҺпјҡ

еҰӮжһңжҢүд»ҘдёҠж–№жі•и®Ўз®—еӨ§ж•ҙж•°зҡ„д№ҳжі•пјҢж—¶й—ҙеӨҚжқӮеәҰжҳҜ O(N2)гҖӮ

дҪҶжҳҜпјҢжҲ‘д»¬жіЁж„ҸеҲ°пјҢеҗ‘йҮҸ {ck} жҳҜеҗ‘йҮҸ {ai} е’Ңеҗ‘йҮҸ {bj} зҡ„еҚ·з§ҜгҖӮ

ж №жҚ®еҚ·з§Ҝе®ҡзҗҶпјҢеҗ‘йҮҸеҚ·з§Ҝзҡ„зҰ»ж•ЈеӮ…йҮҢеҸ¶еҸҳжҚўжҳҜеҗ‘йҮҸзҰ»ж•ЈеӮ…йҮҢеҸ¶еҸҳжҚўзҡ„д№ҳз§ҜгҖӮдәҺжҳҜпјҢжҲ‘д»¬еҸҜд»ҘжҢүз…§д»ҘдёӢжӯҘйӘӨжқҘи®Ўз®—еӨ§ж•ҙж•°д№ҳжі•пјҡ
еҲҶеҲ«жұӮеҮәеҗ‘йҮҸ {ai} е’Ңеҗ‘йҮҸ {bj} зҡ„зҰ»ж•ЈеӮ…йҮҢеҸ¶еҸҳжҚў {Ai} е’Ң {Bj}гҖӮе°Ҷ {Ai} е’Ң {Bj} йҖҗйЎ№зӣёд№ҳеҫ—еҲ°еҗ‘йҮҸ {Ck}гҖӮеҜ№ {Ck} жұӮзҰ»ж•ЈеӮ…йҮҢеҸ¶йҖҶеҸҳжҚўпјҢеҫ—еҲ°зҡ„еҗ‘йҮҸ {ck} е°ұжҳҜеҗ‘йҮҸ {ai} е’Ңеҗ‘йҮҸ {bj} зҡ„еҚ·з§ҜгҖӮеҜ№зҡ„еҗ‘йҮҸ {ck} иҝӣиЎҢйҖӮеҪ“зҡ„иҝӣдҪҚе°ұеҫ—еҲ°дәҶеӨ§ж•ҙж•° a е’Ң b зҡ„д№ҳз§Ҝ cгҖӮ

еҜ№дәҺеӨҚж•°еҗ‘йҮҸ { xN-1, …, x1, x0 }пјҢзҰ»ж•ЈеӮ…йҮҢеҸ¶еҸҳжҚўе…¬ејҸдёәпјҡ

зҰ»ж•ЈеӮ…йҮҢеҸ¶йҖҶеҸҳжҚўе…¬ејҸдёәпјҡ

жіЁж„ҸеҲ°зҰ»ж•ЈеӮ…йҮҢеҸ¶йҖҶеҸҳжҚўйҷӨдәҶжҢҮж•°зҡ„з¬ҰеҸ·зӣёеҸҚд»ҘеҸҠз»“жһңйңҖиҰҒд№ҳд»ҘеҪ’дёҖеҢ–еӣ еӯҗ 1/N еӨ–пјҢдёҺзҰ»ж•ЈеӮ…йҮҢеҸ¶еҸҳжҚўжҳҜзӣёеҗҢзҡ„гҖӮжүҖд»Ҙи®Ўз®—зҰ»ж•ЈеӮ…йҮҢеҸ¶еҸҳжҚўзҡ„зЁӢеәҸзЁҚеҒҡдҝ®ж”№д№ҹеҸҜд»Ҙз”ЁдәҺи®Ўз®—йҖҶеҸҳжҚўгҖӮ

еңЁжҲ‘д»¬зҡ„дҫӢеӯҗдёӯпјҢд№ҳз§Ҝ c = 292896пјҢе…ұ 6 дҪҚж•°еӯ—пјҢN йңҖиҰҒжү©еұ•еҲ° 23 = 8гҖӮйӮЈд№ҲпјҢеҗ‘йҮҸ {ai} е’Ңеҗ‘йҮҸ {bj} еҰӮдёӢжүҖзӨәпјҡ

дёәдәҶжұӮеҮәд»ҘдёҠеҗ‘йҮҸзҡ„зҰ»ж•ЈеӮ…йҮҢеҸ¶еҸҳжҚўпјҢжҲ‘д»¬д»Ө

дёәдәҶж–№дҫҝи®Ўз®—пјҢжҲ‘д»¬йў„е…ҲжұӮеҮә Пү зҡ„еҗ„ж¬Ўж–№пјҢеҰӮдёӢпјҡ

жіЁж„ҸеҲ°еҪ“ n > 2 ж—¶пјҢan = 0пјҢдәҺжҳҜпјҡ

еҗҢж ·пјҢеҪ“ n > 2 ж—¶пјҢbn = 0пјҢдәҺжҳҜпјҡ

иҝҷж ·пјҢеҗ‘йҮҸ {ai} е’Ңеҗ‘йҮҸ {bj} зҡ„зҰ»ж•ЈеӮ…йҮҢеҸ¶еҸҳжҚў {Ai} е’Ң {Bj} еҰӮдёӢжүҖзӨәпјҡ

зҺ°еңЁпјҢе°ҶеҘ№д»¬йҖҗйЎ№зӣёд№ҳеҫ—еҲ°еҗ‘йҮҸ {Ck}пјҢеҚі

дёәдәҶжұӮеҮәеҗ‘йҮҸ {Ck} зҡ„зҰ»ж•ЈеӮ…йҮҢеҸ¶йҖҶеҸҳжҚўпјҢжҲ‘д»¬д»Ө

дёәдәҶж–№дҫҝи®Ўз®—пјҢжҲ‘д»¬йў„е…ҲжұӮеҮә Пү зҡ„еҗ„ж¬Ўж–№(жіЁж„Ҹ Пүk+8 = Пүk)пјҢеҰӮдёӢпјҡ

дәҺжҳҜпјҡ

иҝҷж ·пјҢжҲ‘д»¬е°ұдҪҝз”ЁзҰ»ж•ЈеӮ…йҮҢеҸ¶еҸҳжҚўе’ҢйҖҶеҸҳжҚўи®Ўз®—еҮәдәҶеҗ‘йҮҸ {ai} е’Ңеҗ‘йҮҸ {bj} зҡ„еҚ·з§Ҝеҗ‘йҮҸ {ck}пјҢеҰӮдёӢжүҖзӨәпјҡ

иҝҷе’ҢжҲ‘д»¬еңЁеүҚйқўзӣҙжҺҘдҪҝз”Ёеҗ‘йҮҸ {ai} е’Ңеҗ‘йҮҸ {bj} жқҘи®Ўз®—еҚ·з§Ҝзҡ„з»“жһңжҳҜдёҖж ·зҡ„гҖӮ

дҪҶжҳҜпјҢиҝҷдёӘз®—жі•зҡ„ж—¶й—ҙеӨҚжқӮеәҰиҝҳжҳҜ O(N2)гҖӮжҲ‘д»¬з»•дәҶиҝҷд№ҲдёҖеӨ§еңҲпјҢдёҚжҳҜзҷҪиҙ№еҠІдәҶеҗ—пјҹ

зҺ°еңЁе°ұеҲ°дәҶе…ій”®ж—¶еҲ»

е…ій”®еңЁдәҺпјҡзӣҙжҺҘиҝӣиЎҢзҰ»ж•ЈеӮ…йҮҢеҸ¶еҸҳжҚўзҡ„и®Ўз®—еӨҚжқӮеәҰжҳҜ O(N2)гҖӮеҝ«йҖҹеӮ…йҮҢеҸ¶еҸҳжҚўеҸҜд»Ҙи®Ўз®—еҮәдёҺзӣҙжҺҘи®Ўз®—зӣёеҗҢзҡ„з»“жһңпјҢдҪҶеҸӘйңҖиҰҒ O(N logN) зҡ„и®Ўз®—еӨҚжқӮеәҰгҖӮ N logN е’Ң N2 д№Ӣй—ҙзҡ„е·®еҲ«жҳҜе·ЁеӨ§зҡ„гҖӮдҫӢеҰӮпјҢеҪ“ N = 106 ж—¶пјҢеңЁдёҖдёӘжҜҸз§’иҝҗз®—зҷҫдёҮж¬Ўзҡ„и®Ўз®—жңәдёҠпјҢзІ—з•Ҙең°иҜҙпјҢе®ғд»¬д№Ӣй—ҙе°ұжҳҜеҚ з”Ё 30 з§’ CPU ж—¶й—ҙе’ҢдёӨжҳҹжңҹ CPU ж—¶й—ҙзҡ„е·®еҲ«гҖӮ

еҝ«йҖҹеӮ…йҮҢеҸ¶еҸҳжҚўзҡ„иҰҒзӮ№еҰӮдёӢпјҡ

дёҖдёӘз•Ңй•ҝдёә N зҡ„зҰ»ж•ЈеӮ…йҮҢеҸ¶еҸҳжҚўеҸҜд»ҘйҮҚж–°еҶҷжҲҗдёӨдёӘз•Ңй•ҝеҗ„дёә N/2 зҡ„зҰ»ж•ЈеӮ…йҮҢеҸ¶еҸҳжҚўд№Ӣе’ҢгҖӮе…¶дёӯдёҖдёӘеҸҳжҚўз”ұеҺҹжқҘ N дёӘзӮ№дёӯзҡ„еҒ¶ж•°зӮ№жһ„жҲҗпјҢеҸҰдёҖдёӘеҸҳжҚўз”ұеҘҮж•°зӮ№жһ„жҲҗгҖӮиҝҷдёӘиҝҮзЁӢеҸҜд»ҘйҖ’еҪ’ең°иҝӣиЎҢдёӢеҺ»пјҢзӣҙеҲ°жҲ‘д»¬е°Ҷе…ЁйғЁж•°жҚ®з»ҶеҲҶдёәз•Ңй•ҝдёә 1 зҡ„еҸҳжҚўгҖӮд»Җд№ҲжҳҜз•Ңй•ҝдёә 1 зҡ„еӮ…йҮҢеҸ¶еҸҳжҚўе‘ўпјҹе®ғжӯЈжҳҜжҠҠдёҖдёӘиҫ“е…ҘеҖјеӨҚеҲ¶жҲҗе®ғзҡ„дёҖдёӘиҫ“еҮәеҖјзҡ„жҒ’зӯүиҝҗз®—гҖӮиҰҒе®һзҺ°д»ҘдёҠз®—жі•пјҢжңҖе®№жҳ“зҡ„жғ…еҶөжҳҜеҺҹе§Ӣзҡ„ N дёә 2 зҡ„ж•ҙе№Ӯж¬ЎйЎ№пјҢеҰӮжһңж•°жҚ®йӣҶзҡ„з•Ңй•ҝдёҚжҳҜ 2 зҡ„е№Ӯж¬Ўж—¶пјҢеҲҷеҸҜж·»дёҠдёҖдәӣйӣ¶еҖјпјҢзӣҙеҲ° 2 зҡ„дёӢдёҖе№Ӯж¬ЎгҖӮеңЁиҝҷдёӘз®—жі•дёӯпјҢжҜҸйҖ’еҪ’дёҖж¬ЎйңҖ N йҳ¶иҝҗз®—пјҢе…ұйңҖиҰҒ log N ж¬ЎйҖ’еҪ’пјҢжүҖд»Ҙеҝ«йҖҹеӮ…йҮҢеҸ¶еҸҳжҚўз®—жі•зҡ„ж—¶й—ҙеӨҚжқӮеәҰжҳҜ O(N logN)гҖӮ

з”ұдәҺеҝ«йҖҹеӮ…йҮҢеҸ¶еҸҳжҚўжҳҜйҮҮз”ЁдәҶжө®зӮ№иҝҗз®—пјҢеӣ жӯӨжҲ‘д»¬йңҖиҰҒи¶іеӨҹзҡ„зІҫеәҰпјҢд»ҘдҪҝеңЁеҮәзҺ°иҲҚе…ҘиҜҜе·®ж—¶пјҢз»“жһңдёӯжҜҸдёӘз»„жҲҗйғЁеҲҶзҡ„еҮҶзЎ®ж•ҙж•°еҖјд»ҚжҳҜеҸҜиҫЁи®Өзҡ„гҖӮй•ҝеәҰдёә N зҡ„ B иҝӣеҲ¶ж•°еҸҜдә§з”ҹеӨ§еҲ° B2N йҳ¶зҡ„еҚ·з§ҜеҲҶйҮҸгҖӮжҲ‘д»¬зҹҘйҒ“пјҢеҸҢзІҫеәҰжө®зӮ№ж•°зҡ„е°ҫж•°жҳҜ 53 дёӘдәҢиҝӣдҪҚпјҢжүҖд»Ҙпјҡ

2 x log2B + log2N + еҮ дёӘ x log2log2N < 53

дёҠејҸдёӯе·Ұиҫ№жңҖеҗҺдёҖйЎ№жҳҜдёәдәҶеҝ«йҖҹеӮ…йҮҢеҸ¶еҸҳжҚўзҡ„иҲҚе…ҘиҜҜе·®гҖӮ

жүҖд»ҘпјҢдёәдәҶиғҪеӨҹи®Ўз®—е°ҪйҮҸеӨ§зҡ„ж•ҙж•°пјҢдёҖиҲ¬ B дёҚдјҡеҸ–еҫ—еӨӘеӨ§гҖӮеңЁи®Ўз®—жңәзЁӢеәҸдёӯз»ҸеёёдҪҝз”Ё 256 иҝӣеҲ¶иҝӣиЎҢиҝҗз®—гҖӮдҪҶжҳҜеҰӮжһңз»ҸеёёйңҖиҰҒе°Ҷи®Ўз®—з»“жһңе’ҢеҚҒиҝӣеҲ¶дә’зӣёиҪ¬жҚўпјҢеҲҷеҫҖеҫҖдҪҝз”Ё 100 иҝӣеҲ¶иҝӣиЎҢиҝҗз®—гҖӮ

еҝ«йҖҹеӮ…йҮҢеҸ¶еҸҳжҚўпјҡ

FFTз®—жі•и®ҫи®Ўзҡ„еҹәжң¬жҖқжғіпјҢе°ұжҳҜе……еҲҶеҲ©з”ЁDFTзҡ„е‘ЁжңҹжҖ§е’ҢеҜ№з§°жҖ§пјҢеҮҸе°‘йҮҚеӨҚзҡ„и®Ўз®—йҮҸпјӣе№¶жҠҠNзӮ№й•ҝеәҸеҲ—еҲҶжҲҗеҮ дёӘзҹӯеәҸеҲ—пјҢеҮҸе°‘жҜҸдёӘеәҸеҲ—й•ҝеәҰпјҢеҸҜеӨ§еӨ§еҮҸе°‘и®Ўз®—йҮҸгҖӮ
е®һи·өдёӯдҪҝз”ЁжңҖеӨҡзҡ„FFTжҳҜвҖңеҹә2вҖқз®—жі•гҖӮ
жүҖи°“вҖңеҹә2вҖқпјҢе°ұжҳҜд»ӨDFTзҡ„зӮ№ж•°Nж»Ўи¶і пјҲ5пјү
FFTеҹә2з®—жі•еҲҶдёәж—¶еҹҹжҠҪеҸ–жі•пјҲDecimation In Timeпјүе’Ңйў‘еҹҹжҠҪеҸ–жі•пјҲDecimation In FrequencyпјүдёӨеӨ§зұ»гҖӮ

жң¬ж–ҮйҮҚзӮ№д»Ӣз»Қе…¶дёӯзҡ„ж—¶еҹҹжҠҪеҸ–жі•еҝ«йҖҹеӮ…йҮҢеҸ¶еҸҳжҚўпјҲDITпјҚFFTпјүпјҢз®—жі•и®ҫи®ЎжҖқжғіиҰҒзӮ№еҰӮдёӢпјҡ
1гҖҒжҠҠй•ҝеәҰдёәNзҡ„ж—¶еҹҹеәҸеҲ—xпјҲnпјүжҢүnзҡ„еҘҮеҒ¶еҲҶдёәдёӨз»„пјҢеҸҳжҲҗдёӨдёӘеәҸеҲ—пјҢй•ҝеәҰеқҮдёәN/2гҖӮ
еҚі пјҲ6пјү
е…¶дёӯдёҖдёӘN/2зӮ№зҡ„DFTдёә пјҲ7пјү
еҸҰдёҖдёӘN/2зӮ№зҡ„DFTдёә пјҲ8пјү
2гҖҒдёҚйҡҫжҺЁеҮәеҺҹеәҸеҲ—xпјҲnпјүзҡ„N/2зӮ№DFTдёә пјҲ9пјү

жіЁж„ҸпјҡдёҠејҸд»…жҳҜXпјҲkпјүзҡ„еүҚдёҖеҚҠеҚіN/2зӮ№иҝҗз®—пјҢж•ҙдёӘNзӮ№DFTз»“жһңиҝҳиҰҒеҠ дёҠеҗҺдёҖеҚҠи®Ўз®—гҖӮеҰӮжһңиҖҒиҖҒе®һе®һи®Ўз®—еҗҺдёҖеҚҠN/2зӮ№DFTпјҢеҲҷе№¶жІЎжңүеҮҸе°‘д»»дҪ•и®Ўз®—йҮҸгҖӮдҪҶиҖғиҷ‘еҸҜеҲ©з”ЁDFTеҸҠе…¶еҸҳжҚўеӣ еӯҗзҡ„е‘ЁжңҹжҖ§е’ҢеҜ№з§°жҖ§пјҢе№¶еҲ©з”ЁеүҚдёҖеҚҠи®Ўз®—з»“жһңпјҢеҗҺдёҖеҚҠи®Ўз®—еҸҜиЎЁзӨәдёә пјҲ10пјү
