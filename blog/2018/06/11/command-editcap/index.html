拆分

使用wireshark自带的editcap。我们的系统Centos 5.8，执行 yum install wireshark，就已经安装了editcap。

1. 根据时间来拆分，利用-A 起始时间和-B 截止时间来提去某个时间段的数据。

用法：editcap -A <起始时间> -B <截止时间> <源文件名> <目的文件名>
