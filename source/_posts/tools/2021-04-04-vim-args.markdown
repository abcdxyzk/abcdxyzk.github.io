#### 多文件替换(arg) 

a、加入要处理的文件
```
	:args *.txt
```

b、输入对上述文件的动作
```
	:argdo %s/hate/love/gc | update  （这里将hate替换成love，update表示要写入到文件中，否则只作替换而不写入）
```
